# Stocker_Pro

An AI-powered stock analysis web app. Enter up to 3 US stock tickers and get a 3-day historical data summary with a buy, hold, or sell recommendation generated by Google Gemini.


**Live Demo:** https://stocker-pro.vercel.app/

---

## Tech Stack

**Backend**
- Node.js + Express v5
- [Polygon.io](https://polygon.io/) — historical market data (OHLC)
- [Google Gemini](https://ai.google.dev/) (`gemini-2.5-flash`) — AI analysis

**Frontend**
- Vanilla JavaScript (no frameworks)
- HTML5 + CSS3 (custom properties, flexbox)

---

## Getting Started

### Prerequisites
- Node.js v18+
- A [Polygon.io](https://polygon.io/) API key (free tier)
- A [Google AI Studio](https://aistudio.google.com/) API key (free tier)

### Installation

```bash
git clone https://github.com/your-username/stocker-pro.git
cd stocker-pro
npm install
```

### Environment Variables

Copy `.env.example` to `.env` and fill in your API keys:

```bash
cp .env.example .env
```

```
GEMINI_API_KEY=your_gemini_api_key_here
POLYGON_API_KEY=your_polygon_api_key_here
```

### Running the App

```bash
# Production
npm start

# Development (hot reload)
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## How It Works

1. **Input** — Enter 1–3 US stock ticker symbols (e.g. AAPL, MSFT, TSLA)
2. **Fetch** — The server queries the Polygon.io API for the 3 most recent trading days of OHLC data per ticker
3. **Analyse** — All data is sent to Google Gemini in a single prompt; the model returns a plain-English report with a buy, hold, or sell stance for each ticker

---

## Limitations

| Limitation | Detail |
|---|---|
| Polygon free tier | 5 API calls per minute — hence the 3-ticker cap |
| Gemini free tier | Rate-limited; rapid or concurrent requests may be throttled |
| Data window | 3 trading days only — short-term signal, not comprehensive analysis |
| Data delay | Polygon free tier data is not real-time |
| US tickers only | 1–5 uppercase letters, US exchange-listed equities |
